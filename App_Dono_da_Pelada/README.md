## Ao iniciar a Fase 1 (Antigravity → Claude Code)

1. Inicializar o repositório git **local** aqui (ou na raiz `Projeto_Final_Dev_Web/`) com as 3 branches
   (`AGENTS.md §2`). Sem rede até o portão de deploy.
2. **Copiar `../CLAUDE.md` e `../AGENTS.md` para esta pasta** (raiz que o Claude Code lê) e mantê-las
   sincronizadas.
3. Abrir o **Figma** (link em `../CLAUDE.md §2`) e mapear todas as telas.
4. Criar a fundação **Expo + TypeScript + Drizzle/SQLite** (ver `../Metodo_Modo_Operante_v2/ADR/`).
5. Estrutura sugerida: `src/{screens,components,auth,db,api,lib,theme}/`, `tests/`, `docs/modulos/`.
6. Preencher o **mapa de módulos** em `../MAP.md §4` a cada módulo novo.

> Stack, padrões e Common Hurdles: `../CLAUDE.md`. Fluxo e fases: `../Modo_Operante.md`. Segurança e
> testes: `../Metodo_Modo_Operante_v2/SECURITY.md` e `TESTING.md`.
